🔥 Firebase + Backend Integration (Simple)

ทดสอบการเชื่อมต่อ Firebase Authentication กับ Backend API + Account Linking

📋 Setup Instructions:

  1. Deploy Functions: firebase deploy --only functions
  2. Firebase Config: ตรวจสอบ Firebase Console settings
  3. Authorized Domains: เพิ่ม localhost และ 127.0.0.1
  4. Google OAuth: เปิดใช้งานใน Firebase Console
  5. LINE Login: เปิดใช้งานใน Firebase Console
  6. Apple Sign In: เปิดใช้งานใน Firebase Console

🔗 การเชื่อมต่อ Provider จริง

สำหรับการเชื่อมต่อ provider จริง ต้องตั้งค่า OAuth ใน Firebase Console:

  • Google: เปิดใช้งาน Google provider และตั้งค่า OAuth client
  • Apple: เปิดใช้งาน Apple provider และตั้งค่า Apple Sign-In
  • LINE: เปิดใช้งาน LINE provider และตั้งค่า LINE Login

หมายเหตุ: ต้องล็อกอินก่อนจึงจะสามารถเชื่อมต่อ provider อื่นๆ ได้
